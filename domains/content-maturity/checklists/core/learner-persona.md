# Checklist: Learner personas (didactic “beta” — not literary)

Fixed personas for Fach-/Lehrtext. Tag findings `didaktik` and optional `persona:P1` / `P2` / `P3`.

## P1 — Erstleser:in unter Zeitdruck (30–60 s)

- [ ] In under a minute: is the **main action or takeaway** findable?
- [ ] Gatekeeper / danger / “what now?” not buried under background?
- [ ] Noise (links, history, niceties) de-emphasized where action matters?
- [ ] Absolute safety language either absent or clearly scoped?

→ If fail: severity usually `should` or `block` (if action unfindable in emergency-like genre).

## P2 — Prüfungsvorbereiter:in

- [ ] Stated or implied learning goals map to body sections?
- [ ] Critical Claims exist and are **testable**?
- [ ] Terms needed for exam-style recall introduced before use?
- [ ] Redundant waffle that dilutes “what to remember” flagged?

→ Typical tags: `didaktik`, `claims`, `struktur`.

## P3 — Wiedereinsteiger:in

- [ ] Jargon expanded or glossed on first use?
- [ ] Progressive disclosure: core before deep excursus?
- [ ] Section headings scannable for “where was that fact?”
- [ ] Cross-references not assuming continuous linear read?

→ Typical severity: `should` / `nit`.

## Explicit non-goals

- Not literary taste panels  
- Not market comps / “would you buy this novel?”  
- Not inventing missing clinical content to satisfy a persona

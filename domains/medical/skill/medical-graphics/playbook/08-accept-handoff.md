# Playbook 08 — Decision Note & Accept handoff (G4)

## Decision Note

Fill `domains/medical/templates/graphic-decision.template.md` → `…/<slug>.graphics.md`:

- form_spec_path  
- Phase A free + claim audit verdict  
- Multi-round log (R1–R3)  
- graphic_mode + realize_path per unit  
- Asset manifest  
- Recommended winner  
- Compose embed path  

## Ready for Media Accept?

Only when:

- [ ] Form Spec exists (ambitious units)  
- [ ] Claim audit recorded  
- [ ] At least one production-eligible carrier  
- [ ] Spikes compile  
- [ ] Caps respected  

## Media Accept inputs

- Brief + Form Spec + graphics.md + spikes  
- Media picks winner by **ideal proximity + learner job**  
- Compose notes: which function/asset to embed  

## After Accept

Tech (`compose-chapter`): embed winner; do not re-author ideal.

# Playbook 05 — Hybrid (G3 R2, optional)

## When

Free/ideal needs **form chrome** hard in pure Typst, but claims must stay editable.

## Valid hybrid

Asset (SVG/PNG base) + **Typst labels on node slots** (contiguity).

## Invalid hybrid (never fair A/B)

SVG + **foot legend only** labeled as equal to flowchart.

## Steps

1. Produce base SVG with empty slots or clear regions.  
2. Spike: `image` + `place` / measured layout with claim-safe Typst text on slots.  
3. Same claim pins as code variant.  
4. A/B only against labeled code — not against legend-only.

## Gold lesson

III-2: hybrid v1 unfair → v2 labels on nodes can win ideal proximity; code remains maintainability fallback.

## Cap

1 fair hybrid spike per unit unless Human extends.

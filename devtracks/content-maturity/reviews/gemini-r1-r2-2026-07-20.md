## Final MVP v0.3 scope (consensus)

### In scope

- Transitional kit `domains/content-maturity/`
- Skill `content-review` (core developmental-edit + claims audit + safety-flags)
- Templates: review-report, freeze-record, proof-record (**human**), scope-brief optional
- `content_revision` = `git:…` | `sha256:…`
- Circuit breaker: max 2 C2 rounds
- A: `validate --profile smoke|production` (production → hard freeze + pin match)
- B: `brief_class` + freeze pin for production path
- Synthetic fixtures + gold review → freeze path
- Split checklist / migration notes

### Deferred v0.4+

- Content-patch pipeline (agent diffs)
- Language-proof checklist as first-class
- Full medical pedagogical + clinical-risk profile (guideline depth)
- `genre-hooks.md` (B coupling)
- Agent PDF vision proof
- Separate **Imprimatur** state as hard release gate (optional thin stub allowed in templates for forward-compat)

### Rejected

- Date-based revision pins
- Agent-automated freeze
- Content SoT in A
